📥 Download Files

Everything you need to upload to your GitHub repository for Vercel

How to get this on GitHub + Vercel:
  1. Download all files below using the button at the bottom
  2. Go to your GitHub repo → click Add file → Upload files
  3. Upload server.js, package.json, package-lock.json, and render.yaml to the root of the repo
  4. Create a folder called public and upload all the public/ files into it
  5. Commit the changes — Vercel will auto-redeploy within 30 seconds

Do not upload users.json or sessions.json — those are private and should never go on GitHub.
